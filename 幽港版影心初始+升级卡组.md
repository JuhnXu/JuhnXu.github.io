# 幽港版影心（Shadowheart）

## 职业定位

### 暗影牧誓者（Twilight Cleric）

定位：

* 中距离支援
* 暗影控制
* 治疗
* 诅咒
* 半坦克

难度：★★★☆

风格：
“不是纯奶，而是通过削弱敌人与位置控制来保护队伍。”

---

# 核心机制

## 1. 暗影（Shadow）

部分技能会生成：

* 暗影区块

敌人进入：

* -1攻击
* 或失明
* 或移动减速

类似幽港元素系统。

---

## 2. 信仰（Faith）

某些技能会：

* 获得 Faith

达到2层后：

可强化神术。

例如：

* 群体治疗增强
* 神圣爆发
* 自动护盾

# 最重要的一点

幽港版影心不能做成：

“纯奶妈”。

因为幽港体系里：

* 输出
* 控制
* 生存
* 资源管理

必须同时存在。

真正的强度来源于：

> “让敌人整轮失效。”

这才是幽港风格。

* 手牌节奏

* 先攻
* 烧牌（LOSS）
* 控制价值
* 支援强度

而不是普通RPG技能表。

# 职业：暗月牧誓者（Moonbound Cleric）

## 职业特性

### Faith（信仰）

每回合首次：

* 治疗盟友
  或
* 对敌人施加负面状态

获得1 Faith（最多2）。

部分技能可消耗 Faith 强化。

---

### Shadow（暗影）

部分技能制造“暗影区域”。

暗影区域中的敌人：

* 攻击 -1
* 视线距离 -1

持续1轮。

# 玩法风格总结

影心在这套设计里：

| 类型 | 强度    |
| -- | ----- |
| 治疗 | ★★★   |
| 控制 | ★★★★★ |
| 爆发 | ★★★★  |
| 坦度 | ★★★   |
| 辅助 | ★★★★★ |

真正核心不是奶量。

而是：

* 失明
* 诅咒
* 暗影区域
* 削弱敌人行动

属于标准幽港“减行动经济”型职业。
